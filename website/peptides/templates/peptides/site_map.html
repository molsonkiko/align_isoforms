/peptides?orderby=<len|iso>
: Contains links to all proteins, ordered alphabetically by accession number. Can also add a query to order the proteins by number of isoforms, or by sequence length.

/peptides/proteins/<UniProt accession number>?width=<integer>
: Data for any protein already in the database. Example: /peptides/proteins/P56856. Can change the number of amino acids per line (default 120) with a width query.

/peptides/proteins/<UniProt accession number>
: JSON data for any protein already in the database. Example: /peptides/proteins/P56856/json

/peptides/proteins/json_schema
: JSON schema for the JSON returned by 
peptides/proteins/<accession num>
json.

/peptides/peptides?acc_num_like=<UniProt accession number>
: Download a CSV file with the peptides associated with any protein whose UniProt accession number partially matches the query. Example: /peptides/peptides/acc_num_like=P56856 matches 
P56856
and 
P56856-2
.

/peptides/peptides?acc_num=<UniProt accession number>
: Download a CSV file with the peptides associated with the protein that has exactly that UniProt accession number. Example: /peptides/peptides/acc_num=P56856 matches 
P56856
only.

/peptides/alignments/<list of UniProt accession numbers>?width=<integer>
: The alignment of the proteins with those UniProt accession numbers. Most conveniently accessed from the 
proteins
page for one of those accession numbers. Can change the number of amino acids per line (default 60) with a width query.

/peptides/get_protein
: Enter a UniProt accession number, get its isoforms, peptides, and a sequence alignment with its isoforms.
